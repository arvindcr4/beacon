# Beacon

An AI-first universal email client. One inbox over **Gmail**, **Office 365**,
and any **IMAP** mailbox (Yahoo, AOL, iCloud, Fastmail, custom). Mobile-ready
PWA, installable from the browser.

> Built with Claude Code, the *Agent OS* workflow, spec-first development,
> custom subagents, slash-skills, and PreToolUse/PostToolUse hooks.

## What you get

- **Unified inbox** across all your accounts, with per-message priority
  (Now / Soon / Later) generated by Claude Haiku 4.5.
- **AI summaries** — one sentence telling you what an email actually needs.
- **AI reply drafts** with five tones (neutral, warm, direct, apologetic,
  enthusiastic), streamed token-by-token from Claude Sonnet 4.6.
- **Compose / reply / forward** with cross-thread reference headers.
- **Search, labels, archive, delete** — unified across providers.
- **Encrypted credential vault** — AES-256-GCM at rest, refresh-token
  rotation on every OAuth call.
- **Installable PWA** with service worker and mobile-first UI.

## Quick start

```bash
pnpm install
cp .env.example .env.local && $EDITOR .env.local
pnpm db:push
pnpm dev
```

Generate the encryption key with:

```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```
